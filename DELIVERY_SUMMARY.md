## 🎓 How It Works

### Page Load Flow

1. User visits URL (e.g., http://localhost:5173)
2. index.html loads
3. main.jsx mounts React app
4. App.jsx sets up BrowserRouter
5. Navbar renders (sticky at top)
6. Routes component renders active page
7. Page component loads with data
8. Child components render (cards, etc.)
9. Footer renders (sticky at bottom)
10. CSS animations start
11. All interactive elements have hover effects

### Styling Flow

1. globals.css loads first (base styles)
2. Component CSS files load (specific styles)
3. CSS variables are applied from :root
4. Responsive breakpoints trigger on 768px
5. Hover states trigger on pointer hover
6. Animations run based on @keyframes

### Navigation Flow

1. User clicks link in Navbar
2. React Router changes URL
3. Active page component mounts
4. Page animates in with fadeSlideIn
5. Components render with data
6. Active link in Navbar highlights
