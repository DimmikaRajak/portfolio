## ❓ FAQ

**Q: Can I change the colors?**  
A: Yes! Edit the CSS variables in `src/styles/globals.css`

**Q: How do I add more projects?**  
A: Add objects to the `projects` array in `src/pages/Portfolio.jsx`

**Q: How do I change fonts?**  
A: Update imports in `index.html` and CSS in `globals.css`

**Q: Is the contact form working?**  
A: It uses `mailto:` by default. See CUSTOMIZATION_GUIDE for advanced options.

**Q: Can I add a blog section?**  
A: Yes! Create a new `Blog.jsx` page and add a route in `App.jsx`

**Q: What's the best platform to deploy?**  
A: Netlify (easiest, fastest, most features)

**Q: How do I add my own domain?**  
A: See DEPLOYMENT_GUIDE.md for detailed instructions

**Q: Can I customize the animations?**  
A: Yes! All animations are in `globals.css` - edit the @keyframes
