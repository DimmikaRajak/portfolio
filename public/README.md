# Public Assets Placeholder

This folder should contain:

1. **her-photo.jpg** - Your profile photo (240px wide recommended)
   - Place a 240x310px photo here
   - Will be displayed in the Home page hero section with special styling

2. **resume.pdf** - Your resume/CV document
   - Used for "Download CV" button links
   - Visible from Navbar and Home page

## Setup Instructions

1. Replace `her-photo.jpg` with your actual profile photo
2. Replace `resume.pdf` with your actual resume
3. The portfolio will automatically serve these files from the public folder

## Photo Recommendations

- **Format**: JPG or PNG
- **Size**: 240px × 310px (optimal)
- **Quality**: High quality, professional photo
- **Styling**: The photo card has an asymmetric border-radius (28px 28px 80px 28px) and will be displayed with a gradient overlay showing your name and role.

## Resume Recommendations

- **Format**: PDF
- **Pages**: 1-2 pages recommended
- **Size**: Keep file size under 2MB for fast downloads
